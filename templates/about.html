About the Project

Space Station Safety Objects Detection is a cutting-edge project aimed at enhancing safety in Space Station by automatically detecting the necessary protective equipment such as OxygenTank ,NitrogenTank ,FirstAidBox,FireAlarm,SafetySwitchPanel,EmergencyPhone,FireExtinguisher. Our system uses advanced computer vision and deep learning techniques to identify objects in real-time images, and maintain compliance with safety protocols.

Features

  • Automatic detection of safety equipment in images.
  • Confidence score for each detected item.
  • Detection history with visual results.
  • Interactive and easy-to-use web interface.

Technology Used

  • Python & Flask for web application.
  • Synthetic dataset from Falcon.
  • YOLOv8 deep learning model for object detection.
  • OpenCV for image processing and annotation.
  • HTML, CSS, and JavaScript for frontend design.
